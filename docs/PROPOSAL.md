# IP Weave - 项目提案

## 问题

链上 IP 资产在发行后普遍面临内容运营断层。大量 NFT 项目在 mint 结束后缺乏持续的内容生产能力，社区成员受限于专业技能难以参与衍生创作，自发产出的内容风格参差不齐。这导致链上 IP 陷入沉睡，其文化和商业价值远未被充分挖掘。

具体表现为三个核心问题：第一，内容生产门槛高，需要作家、编剧、分镜师、设计师、3D 建模师、开发者等多个角色协同工作，成本高周期长。第二，风格不统一，不同创作者对同一 IP 的理解不同，产出的内容缺乏一致性。第三，缺乏变现路径，衍生内容创作后缺少上链渠道。

## 解决方案

IP Weave 是一个基于 GLM-5.1 的自治 AI Agent，用 AI 替代整个人类创作团队。用户只需要输入一个链上 IP 名称或合约地址，Agent 自主完成从数据读取、风格分析、内容创作到上链交付的完整流程。

核心创新在于 ReAct 架构让 GLM-5.1 在每个循环中真实地分析状态、做出决策，而不是按固定顺序执行函数。同时 Agent 在启动时会先进行任务拆解，制定多步骤计划，然后按计划执行并在每轮评估进度。

## 目标用户

NFT 项目方和 DAO：需要自动化 IP 内容运营，降低人力成本，为社区持续产出高质量内容。

链上 IP 持有者：希望为自己的数字资产生成专属衍生内容，提升资产价值。

Web3 内容创作者：需要获取创作灵感和素材，加速创作流程。

Meme 社区：需要为社区 IP 持续产出新鲜内容，保持社区活跃度。

## 技术实现

系统使用 GLM-5.1 作为核心 LLM 驱动所有决策和创作。web3.py 用于以太坊链上数据读取。CogView-4 用于图片生成。build123d 用于 3D 参数化建模。solc 用于合约编译。httpx 用于外部 API 调用。

系统采用 Agent 加工具的架构。Agent 是总指挥，启动时由 GLM-5.1 分析任务并将其拆解为多个步骤。执行过程中每轮读取当前状态，决定下一步调用哪个工具。执行后评估结果，判断是否需要迭代修正。工具层当前注册了八个独立的功能模块，分别负责链上数据读取、风格分析、故事创作、脚本生成、资产设计、3D 建模、质量检查和交付。

数据流向为：Agent 收到 IP 名称后，调用 read_ip 工具从以太坊主网读取链上数据，将结果存入状态对象。然后调用 analyze_style 工具让 GLM-5.1 分析风格特征。接着依次调用 write_story、write_script、design_assets 生成内容。如有需要调用 render_3d 生成 3D 模型。全部内容生成完毕后调用 check_quality 检查风格一致性。最后调用 deliver 生成 HTML 报告和 NFT 合约。

## 当前完成度

已完成 GLM-5.1 API 调用封装，含超时重试和错误处理。已完成链上 IP 数据实时读取，支持以太坊主网 ERC-721 合约。已完成 IP 风格分析，能提取每个 IP 独有的 lore、角色性格和代表符号。已完成衍生故事创作，能达到 3000 字以上的完整叙事。已完成动画分镜脚本生成，6 到 8 个专业分镜。已完成周边资产概念设计，含 NFT 元数据。已完成 3D 参数化建模，使用 build123d 生成 GLB 格式模型。已完成风格一致性检查，低于 7 分自动迭代修正。已完成 NFT 元数据打包和 Solidity 合约生成。已完成 ReAct Agent 自主决策循环，含任务拆解和多步骤计划。合约已部署到 Sepolia 测试网，有链上证明。

## 后续计划与商业前景

### 产品迭代路线图

第一阶段（1-2 个月）：接入 CogVideoX 或 Seedance 2.0 实现视频动画生成，让 Agent 能直接产出动画短片。增加更多内容类型，如音乐生成和 3D 场景包。优化故事质量和分镜精度。

第二阶段（2-3 个月）：构建 Web 界面，降低非技术用户使用门槛。支持拖拽式操作和一键部署。接入更多链上协议，扩展 IP 来源范围。添加批量 IP 处理功能。

第三阶段（3-6 个月）：支持 Solana 等非 EVM 链部署。构建创作者经济平台，允许用户通过 Agent 创作并上链变现。

### 商业计划

SaaS 订阅模式：为 NFT 项目方提供月度内容运营服务，按 IP 数量和使用频次收费。包含自动化故事生成、脚本创作、资产设计和链上部署全套服务。

单次创作服务：为个人持有者提供单次衍生内容生成服务。输入 IP 名称即可获得完整的衍生内容包。

技术授权模式：将 Agent 能力封装为 API，授权给其他 Web3 平台和市场使用。

链上分润模式：通过 RevenueSplitter 合约实现创作者、IP 持有者、平台之间的自动分润。

### 竞争优势

全链路闭环：目前市场上唯一能同时完成故事创作、分镜脚本、资产设计、3D 模型生成并自动上链的产品。

成本优势：传统内容生产需要 5-10 人团队协作 2-4 周，成本 5-20 万元。IP Weave 将成本降低到几乎为零。

风格统一：GLM-5.1 深度分析每个 IP 的风格特征，在每步创作前都检查风格一致性。
