关于项目

IP Weave 是一个基于 GLM-5.1 的自治 AI Agent，专为链上 IP 衍生内容创作设计。它以 ReAct 架构为核心，启动时先由 GLM-5.1 自主拆解任务并制定多步骤计划，随后逐轮分析当前状态、调用合适的工具执行、评估结果并迭代修复。Agent 集成了链上数据读取、风格分析、故事创作、脚本生成、资产设计、3D 建模、质量检查和合约部署八个工具，可自动完成从创意输入到链上交付的完整 Web3 工作流。

一人加一 Agent，替代传统需要整个创作团队才能完成的 IP 衍生内容生产工作。项目已部署到 Sepolia 测试网，有可验证的链上合约。

自主工作流程

GLM-5.1 在每个决策点分析当前状态后自主决定下一步，不是按固定顺序执行。

1读取链上 IP 数据
2GLM-5.1 分析风格
3GLM-5.1 创作故事
4GLM-5.1 生成分镜脚本
5设计周边资产概念
63D 模型生成
7GLM-5.1 质量检查
8部署合约并交付

工具模块

点击查看每个工具的详细信息。

ip_reader
从以太坊主网读取 ERC-721 链上数据
通过 web3.py 读取合约的 name、symbol、tokenURI。支持通过 IP 名称自动查找合约地址，也支持直接输入合约地址。读取到的元数据作为后续所有步骤的基础。
style_analyzer
GLM-5.1 提取 IP 风格指纹和文化背景
GLM-5.1 根据链上数据提取 IP 的视觉风格、叙事基调、角色性格和代表符号。每个 IP 分析结果都是独特的，包含 lore、character、key_symbols 等信息。
story_writer
GLM-5.1 创作 2000-4000 字衍生故事
基于 IP 独有的世界观和角色设定创作完整叙事。要求完整故事弧线（起因-发展-转折-高潮-收束），场景有画面感，主题有深度。
script_writer
GLM-5.1 生成 6-8 个分镜的动画脚本
将故事转化为专业分镜脚本，每个分镜包含场景描述、对白内容、镜头运动方式和持续时间。可直接用于动画制作。
asset_designer
设计 2 款可上链的周边资产概念
每款资产包含产品名称、类型、设计描述、设计理念和 NFT 元数据。支持调用 CogView-4 生成概念图。
render_3d
用 GLM-5.1 生成 3D 展示页面
GLM-5.1 根据资产描述生成 Three.js 3D 展示 HTML 页面，含轨道控制器、材质光照和动画效果。不需要外部 API。
quality_checker
GLM-5.1 检查风格一致性
评估内容与 IP 风格画像的匹配度，评分 0-10 分。7 分以上通过，未通过自动触发迭代修正。评估维度包括视觉风格、叙事基调和角色设定一致性。
deliver
生成 HTML 报告和 NFT 合约
生成 story.html、script.html、assets.html、index.html 等交付物。打包 NFT 元数据，生成 Solidity ERC-721 合约，准备部署到测试网。
