项目简介

IP Weave 是一个基于 GLM-5.1 的自治 AI Agent，专为链上 IP 衍生内容创作设计。它以 ReAct 架构为核心，启动时先由 GLM-5.1 自主拆解任务并制定多步骤计划，随后逐轮分析当前状态、调用合适的工具执行、评估结果并迭代修复。

一人加一 Agent，替代传统需要整个创作团队才能完成的工作。给一个链上 IP，它还你一个世界。

工作流程

1. 读取链上 IP 数据 — 从以太坊主网获取 IP 元数据

2. 分析风格 — GLM-5.1 提取 IP 视觉和叙事特征

3. 创作故事 — 生成 2000-4000 字完整叙事

4. 生成脚本 — 转化为 6-8 个分镜的专业动画脚本

5. 设计资产 — 周边资产概念和 3D 模型

6. 检查质量 — 风格一致性评分，自动迭代修正

7. 上链交付 — 生成 NFT 合约并部署到测试网
